Orchestrate Your AI Workforce

LLM Ensemble is a command-line tool designed for software engineers and researchers who need higher quality answers than a single AI model can provide.

By running multiple models (Gemini, Codex) in parallel or iteratively, and then using a "Judge" model to synthesize the results, LLM Ensemble reduces hallucinations, catches logic errors, and provides comprehensive solutions.

Installation & Requirements

Prerequisites Ensure you have Python 3.8+ installed. You also need the proprietary gemini and codex CLI tools available in your system PATH.

No pip installation is required for the tool itself. It uses the standard library.

# 1. Check Python version
python3 --version

# 2. Verify external tools
which gemini
which codex

# 3. Make the script executable (optional)
chmod +x llm_ensemble.py

Quick Start

New! Graphical Interface Prefer a visual experience? Run python3 llm_ensemble_gui.py to launch the new GUI.

The most basic usage requires defining the models to use (-m) and the number of iterations (-n).

python3 llm_ensemble.py -m gemini,codex -n 1 "Why is the sky blue?"

What just happened?

  1. The tool ran Gemini once.
  2. The tool ran Codex once (in parallel).
  3. It gathered both answers.
  4. It used a "Merger" model (Codex) to combine them into one final explanation.
  5. It saved everything in a timestamped folder Outputs/llm_ensemble_YYYYMMDD_.../.

Common Scenarios

Scenario 1: The Code Review

You want multiple opinions on a snippet of code to find bugs.

python3 llm_ensemble.py \
  -m gemini,codex \
  -n 2 \
  -f buggy_function.py

Outcome: You get 4 total critiques (2 from Gemini, 2 from Codex). The final output synthesizes all identified bugs into a clean list.

Scenario 2: Analyzing Documents (PDF/DOCX)

The tool can natively read PDFs and Word documents as context.

python3 llm_ensemble.py \
  -m gemini \
  -n 1 \
  -c contract_draft.docx \
  -c legal_precedent.pdf \
  "Compare the draft against the precedent."

Outcome: Extracts text from the DOCX and PDF, attaches them to the prompt, and generates a comparison.

Scenario 3: Generating Documentation (RTF)

Need a formatted document to share with a manager? Output directly to RTF.

python3 llm_ensemble.py \
  -m gemini \
  -n 1 \
  --format rtf \
  "Draft a project proposal for a new API"

Outcome: A final.rtf file is created in the output directory, ready to open in Word or TextEdit.
